Menu Vaucanson

Site de gestion de menu pour mon restaurant scolaire
➜ Voir sur GitHub

mon rôle

j'ai été responsable du front-end, j'ai notamment travaillé sur l'interface administrateurs, elle devait permettre d'analyser des statistiques.

le projet

ce projet a été réalisé en deux semaines, il a été réalisé en binôme, ces sites avaient pour objectif de remplacer les menus papier affichés souvent en retard. nous avons pu conventionner avec le restaurant scolaire pour obtenir les menus à l'avance, nous avions aussi pour but de récolter des statistiques sur l'appréciation des plats par les étudiant·es
